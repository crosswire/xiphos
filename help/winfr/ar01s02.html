Démarrer Xiphos

Vous pouvez démarrer Xiphos en utilisant l'une des méthodes suivantes :

Menu applications

Sélectionnez Accessoires → Xiphos.

Ligne de commande

Entrez xiphos, puis appuyez sur Entrée.

Windows

Sélectionnez Tous les programmes → Xiphos· → Xiphos.

Démarrer Xiphos la première fois

Au premier démarrage de Xiphos, les options sont initialisées à leurs valeurs par défaut. Par ailleurs, si aucune autre application Sword n'a jamais été utilisée auparavant, et donc qu'aucun module Sword n'a été installé, Xiphos demande la permission de démarrer le gestionnaire de modules afin d'installer un module de textes bibliques dans la langue souhaitée.

Xiphos s'ouvre ensuite sur le texte de Romains 8:28 dans un premier onglet. Une fois la fenêtre affichée, il est possible de modifier les options de Xiphos en sélectionnant Éditer → Préférences.

Démarrer Xiphos

Au démarrage de Xiphos, la fenêtre suivante est affichée :

Figure 4. Interface de Xiphos


La fenêtre de Xiphos contient les éléments suivants :

Barre de menu

La Barre de menu permet d'accéder aux fonctions complémentaires de Xiphos, comme le bloc-notes où la recherche avancée. Elle permet également d'ajuster les préférences de Xiphos.

Barre d'outils

La Barre d'outils contient des boutons permettant de naviguer rapidement dans le texte biblique.

Volet latéral

Le Volet latéral présente plusieurs fonctions. Il contient la liste des modules installés, les signets définis par l'utilisateur et il permet d'exécuter des recherches simples.

Textes bibliques

Le volet des Textes bibliques affiche le texte biblique sélectionné.

Aperçu

Le volet Aperçu affiche les textes complémentaires des modules de textes bibliques. Ces textes comprennent les nombres de Strong, les notes de bas de page et les étiquettes morphologiques. Il est situé en-dessous du volet des textes bibliques.

Commentaire

Le volet Commentaire affiche le commentaire relatif au verset sélectionné.

Livre

Le volet Livre affiche le texte de livres disponibles auprès du Sword Project. Ces modules comprennent des textes comme "Calvin's Institutes", "Josephus: The Complete Works" etc. Ce volet est accessible via l'onglet Livre, en dessous du volet Commentaire.

Dictionnaire

Le volet Dictionnaire situé en-dessous du volet Commentaire affiche la définition des mots sélectionnés dans le volet des Textes bibliques.

Un clic du bouton droit de la souris dans les différents volets de la fenêtre de Xiphos fait apparaître un menu contextuel permettant d'imprimer ou d'accéder aux options spécifiques du module concerné.
