Introduction

Xiphos est un logiciel d'étude biblique utilisant l'architecture logicielle du Projet SWORD et l'environnement de bureau «­­ GNOME ». Xiphos permet d'effectuer les tâches suivantes :

  • Rechercher un verset particulier de l'Écriture

  • Créer des études ou des notes personnelles sur une sélection de passages

  • Suivre automatiquement les renvois des notes de bas de pages et des références croisées

  • Comparer des traductions différentes en parallèle

  • Étudier dans les langues originelles, grec ou hébreu

Xiphos a pour but de présenter une interface utilisateur simple, épurée et de fournir à la fois un outil riche, permettant d'offrir un environnement d'étude biblique personnalisé.

Figure 1. Le logo du Projet SWORD


Le Projet­­ SWORD est hébergé à http://www.crosswire.org. D'autres applications utilisant l'architecture logicielle du Projet­­ SWORD sont : The SWORD Project for Windows (BibleCS ou WinSword), MacSword pour Macintosh, BibleDesktop, une application Java et BibleTime sous environnement KDE pour Linux.
